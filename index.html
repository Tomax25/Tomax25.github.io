Bienvenidos

Esta página fue publicada usando GitHub Pages como parte de una actividad académica.

GitHub es una plataforma que permite almacenar, administrar y compartir proyectos de desarrollo de software utilizando el sistema de control de versiones llamado Git. Su función principal es facilitar el trabajo colaborativo entre desarrolladores, llevando un registro de los cambios realizados en el código.

Con GitHub Pages, GitHub ofrece un servicio de hosting gratuito que permite publicar páginas web estáticas directamente desde un repositorio. Solo es necesario subir los archivos HTML, CSS o JavaScript y activar la opción GitHub Pages para que el sitio esté disponible en Internet.

Esta herramienta es muy utilizada en el ámbito académico y profesional para mostrar proyectos, portafolios y documentación técnica de forma sencilla y eficiente.
